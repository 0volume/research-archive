# TAR - Think And Reflect

**Version:** 1.0
**Created:** 2026-02-18

---

## Purpose
Deep learning and comprehensive analysis of research findings. Where RAPP is discovery, TAR is learning.

---

## Frequency
Daily (recommended: mid-day or early afternoon)

---

## Credit Budget
- **RAPP portion:** 2-5 credits (discovery)
- **TAR extra:** +2 credits (deep analysis)
- **Total:** Up to 7 credits per combined session

---

## The TAR Process

### 1. Select Candidates from Archive
- Review existing research entries
- Pick 2-3 related papers that connect to each other
- Look for patterns across entries

### 2. Deep Dive Search (+2 credits)
- Use Tavily to search for related context
- Find papers that connect the dots
- Verify URLs

### 3. Cross-Reference Analysis
- How do these papers relate?
- What patterns emerge?
- What was missed in initial RAPP scan?

### 4. Generate Deeper Insights
- Write substantive thought (not process update)
- Connect to previous insights
- Identify gaps in knowledge

### 5. Consider Pitches
- Does this deeper analysis suggest a new pitch?
- Does it extend an existing pitch?
- Document pitch potential

### 6. Update Recursively
- Read previous TAR thoughts
- Build on previous analysis
- Knowledge should compound

---

## Output
1. Deep insight thought (TH-###)
2. Potential new/changed pitches
3. WhatsApp notification if significant

---

## Rules
- Must be REAL insight, not process update
- Connect ideas across papers
- Build on previous thoughts
- Reference previous TARs
